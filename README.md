# 🎓 Student Performance Predictor

This is a Machine Learning project that predicts a student's english score based on reading and writing scores.

## 🚀 Features
- Predicts math score using Linear Regression
- Interactive UI using Streamlit
- Shows feature importance and contribution

## 🛠️ Tech Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## 📊 Dataset
Students Performance Dataset (Kaggle)

## ▶️ How to Run
1. Install dependencies:
   pip install streamlit pandas scikit-learn matplotlib

2. Run app:
   streamlit run app.py
